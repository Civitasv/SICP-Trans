* 【翻译】计算机程序的构造与解释

** 为什么

这本书对我的编程思维的影响是巨大的，阅读此书，有一种法师在阅读岁月史书的感觉，感觉自己真正进入了程序设计的领域。

本书使用 Scheme(Lisp 方言)进行程序的演示，其语法极其简洁、优雅却又极其强大，其超时代的程序设计思维体现在现代各种编程语言上。

因此，我希望尽己所能，将本书翻译为中文，供相关领域人员翻阅。

** 频率

视空闲时间决定。
